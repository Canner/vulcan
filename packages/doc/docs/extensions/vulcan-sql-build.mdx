# Extensions of the @vulcan-sql/build package

The @vulcan-sql/serve provide 3 extension components:

| Extensions                                         | Description                                                                                                                                                                |
| -------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Packager](./vulcan-sql-build/packager)            | Beside package to npm and dockerfile, extend Packager to package other type format for publishing                                                                          |
| [SchemaReader](./vulcan-sql-build/schema-reader)   | Beside current `LocalFile` type to read YAML format [API schema](../api-building/configuration/api-schema), extend SchemaReader to provide other reader to read API schema |
| [SpecGenerator](./vulcan-sql-build/spec-generator) | Beside `oas3` (Open API version 3 specification) , extend SpecGenerator to provide other API specification                                                                 |
