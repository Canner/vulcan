# Writing SQL

With VulcanSQL, you can build APIs easily with only SQL and Jinja, a lightweight templating language.

* [SQL Syntax](./sql-syntax): Learn how to write SQL with templating language.
* [Predefined Queries](./predefined-queries): Learn how to predefine some queries first, then reference the query results in your template to construct your SQL.
